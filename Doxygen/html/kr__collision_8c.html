Function to detect the collision of a rectangle with the level and an other rectangle,.

Parameters
pLevela pointer to the level structure ( NULL if you don't want to check )
pRect1a pointer to the rectangle in movements
pRect2a pointer to the second rectangle who is not moving ( NULL if you don't want to check )
vxthe movement vector on X you want to move the rectangle 1
vythe movement vector on Y you want to move the rectangle 1
NewVxthe new movement vector on X, which might get modified by the function, you MUST sent NULL as a parameter
NewVythe new movement vector on Y, which might get modified by the function, you MUST sent NULL as a parameter
Returns
0 : The function did not check any collision 1 : No collision with the rectangle 2 : collision with the rectangle 3 : No collision with the level 6 : collision with the level
